# BRAND DNA EXTRACTOR SKILL
**Status:** ACTIVE
**Tool:** Firecrawl / Scraper

## 1. MISSION
Extract the "Soul" of a brand before building UI or Video.

## 2. EXTRACTION PROTOCOL
Input: URL
Output: Update `root/brand_guidelines.md`

1.  **Scrape:** Get HTML/CSS.
2.  **Extract:**
    * **Hex Codes:** Primary, Secondary, Background.
    * **Fonts:** Family names.
    * **Tone:** Adjectives (e.g., "Playful", "Corporate").
    * **Logo:** Save SVG/PNG URL.

## 3. USAGE
All `vibecoding` tasks MUST read `brand_guidelines.md` first.
